# ADB_SC
ADB usb/wireless connect switcher



## Dependencies

### Install ADB

**Install ADB on Debian**
```sh
sudo apt install adb
```

**Install ADB on Fedora**
```sh
sudo dnf install adb
```

**Install ADB on Arch**
```sh
sudo pacman -Syyu
```
```sh
sudo pacman -S android-tools
```

**Install ADB on MacOS**
```sh
brew install android-platform-tools
```
